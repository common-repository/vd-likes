== Description ==

vd likes is a like or unlike to perticular post ,page or anything!
It provides IP based like unlike system.
You can set only register user can give liking system from admin panel.




== Installation ==

1. Upload the `vd_like` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Use the shortcode `[vd_likes]` in your posts or pages.if you want to use shortcode inside for loop or while lood use : do_shortcode('[vd_likes id='POST_ID']');
